Indonesian SMS Classifier
Scripts to create a model to be used as an SMS classifier. This classifier is uses NLTK for string processing. Scikit-learn, Tensorflow and Keras for text classification.

We couldn't have assembled this rhetoric scoring model without the help of some awesome creators on the internet.
