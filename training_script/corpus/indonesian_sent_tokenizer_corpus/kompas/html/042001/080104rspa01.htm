 RS Panti Wilasa Terbakar, Ratusan Pasien Diungsikan

Kompas/hariadi saptono
Semarang, Kompas - Ratusan pasien Rumah Sakit (RS) Panti Wilasa di Jalan Dr Cipto, Semarang, Sabtu (7/4) malam diungsikan menyusul kebakaran yang melanda RS tersebut. Termasuk di antaranya seorang bayi yang berusia beberapa hari dan masih berada di dalam inkubator, diungsikan ke Gereja Kristen Jawa (GKJ) Jateng Utara.

Kebakaran yang mulai terjadi sekitar pukul 21.30 WIB berasal dari toko roti Marimas yang berdampingan di sebelah utara RS Panti Wilasa. Api yang berkobar di bagian tengah toko, langsung menyambar ruang perawatan di samping ruang unit gawat darurat.

Tiga mobil pemadam kebakaran baru mampu mengendalikan api setelah datang mobil pemadam kebakaran yang menggunakan tangga tinggi. Sekitar pukul 22.00 api sudah bisa dikendalikan. Pemadaman api sulit dilakukan karena yang terbakar bagian tengah RS dan upaya pemadaman terhalang papan reklame di bagian depan. Namun hingga berita ini diturunkan pukul 00.10, api sudah bisa dikendalikan dan mobil pemadam masih terus menyemprotkan air.

Kepanikan di kalangan pasien RS itu tidak bisa dihindarkan. Pasien yang masih mampu berjalan, segera melarikan diri. Beberapa di antaranya digotong, termasuk bayi yang masih berada dalam inkubator. Di GKJ Jateng Utara di depan RS tersebut, perawat dan dokter bekerja keras merawat pasien yang ditampung di sana.

Dalam suasana gerimis, penyelamatan pasien tidak bisa cepat karena di dalam gereja penuh bangku sehingga bangku-bangku tersebut harus dikeluarkan lebih dahulu. Masyarakat sekitar berusaha membantu agar pasien yang berada di halaman gereja segera tertampung.

Menurut keterangan saksi mata kepada Kompas, kejadian begitu cepat dan api berkobar seketika membuat pasien panik. Sedangkan sumber api yang menyebabkan kebakaran masih belum diketahui. Lokasi RS tersebut berdampingan dengan deretan pertokoan. (son/hrd)
