 Musibah Kembang Api di Peru
* 122 Orang Tewas

Lima, Minggu

Sedikitnya 122 orang tewas, 180 luka-luka, dan 40 masih dinyatakan hilang dalam kebakaran hari Sabtu (29/12) yang melanda sebuah wilayah pertokoan di Lima, Peru, yang didahului dengan ledakan di sebuah toko penjual kembang api. Sedangkan di Cina, kebakaran akibat kembang api hari Minggu mengakibatkan sembilan nyawa melayang.

Menurut penyidik Peru, Maria del Pilar Peralta, jumlah korban tewas di Lima kemungkinan masih akan terus bertambah, karena para pekerja penyelamat tak bisa memasuki setiap bangunan pertokoan yang dilalap api, dengan alasan risiko bangunan runtuh.

Kebanyakan korban, menurut para petugas, terbakar sampai mati atau mati akibat sesak napas karena asap, dalam kebakaran yang dilukiskan sebagai tragedi yang menelan korban terburuk yang pernah terjadi di ibu kota Peru, Lima.

Hari Minggu pagi, Presiden Peru Alejandro Toledo-yang sempat membatalkan kunjungan kerjanya dan kemudian mengalihkan kunjungannya ke lokasi musibah-melukiskan musibah itu sebagai "malapetaka mengerikan". Dalam pernyataannya hari Sabtu malam, Toledo menyatakan dua hari berkabung nasional di seluruh negeri, guna menyatakan kesedihannya bagi para korban kebakaran. Hari berkabung di seluruh negeri Peru dimulai sejak Minggu.

Menurut sejumlah saksi mata, api mulai berkobar setelah terjadi kebakaran di sebuah toko yang menjual kembang api buatan sendiri secara ilegal, pada sekitar pukul 07.30 waktu setempat, setelah sejumlah penjual mengetes sejumlah kembang api.

Akhir tahun

Kobaran api di Peru ini setidaknya menghancurkan sebuah bangunan tinggi di lokasi bersejarah Peru itu, sejumlah apartemen tempat tinggal, dan beberapa mobil. Akibat kebakaran juga dirasakan di wilayah empat blok dari lokasi kejadian.

Sejumlah rumah sakit di seantero ibu kota Peru Minggu pagi itu dipenuhi dengan korban-korban yang menderita luka bakar, ataupun menghirup asap. Para petugas paramedis dan kepolisian berupaya melakukan aksi penyelamatan terhadap korban yang masih bertahan hidup, dengan melarikan diri ke arah atap bangunan yang tingginya bisa mencapai lebih dari delapan tingkat.

Petugas-petugas ini dihadapkan pada kesulitan mendapatkan air dari pompa di jalanan, kurang tenaga listrik, dan harus menembus asap hitam tebal yang membuat pekat wilayah itu. Mereka juga dihadapkan kesulitan untuk menuju lokasi karena banyaknya warga yang memadati jalanan sempit di sekitar pusat pertokoan itu.

Api berkobar di sebuah toko pusat kota, yang dipadati dengan pedagang-pedagang kembang api. Jalanan itu biasanya memang dipadati pengunjung, yang membeli kembang api untuk keperluan perayaan akhir tahun.

"Saya menyaksikan sisa-sisa jasad yang terperangkap api di dalam mobil-mobil mereka, serta tubuh-tubuh yang sudah hangus di jalanan," kata Kepala Pemadam Kebakaran Kota Lima Tulio Nicolino.

Sedangkan kebanyakan penjual yang tewas, kata Nicolino, terjebak dalam toko mereka, karena "tak ingin meninggalkan toko mereka", lantaran takut penjarah.

Menurut Mendagri Peru Fernando Rospigliosi, larangan terhadap penjualan kembang api sebenarnya sudah diumumkan, tetapi tidak banyak diindahkan. Rospigliosi berjanji, dalam beberapa waktu mendatang penguasa Lima akan menerapkan sejumlah hukuman berat bagi mereka yang nekat menjual kembang api secara ilegal.

Sebuah tayangan musibah sempat disiarkan langsung, menggambarkan tubuh-tubuh yang hangus terbakar, bertumpukan satu sama lain. Jumlah korban termasuk anak-anak belasan tahun.

Wilayah historis Kota Lima, ibu kota negeri miskin dengan penduduk delapan juta jiwa ini, sudah dinyatakan sebagai sebuah warisan dunia oleh Perserikatan Bangsa-Bangsa.

Di Cina

Kebakaran akibat kembang api di Cina menewaskan setidaknya sembilan orang dan mencederai 40 orang lainnya. Kebakaran bermula dari sebuah ledakan di sebuah pabrik kembang api di Provinsi Jiangxi, yang kemudian melalap 10 dari 11 bangunan di lingkungan pabrik.

Menurut kantor berita Xinhua, perusahaan yang terkena musibah itu adalah perusahaan bernama "Panda", yang memang dikenal mengekspor kembang api produksinya ke berbagai penjuru dunia.

Ledakan yang terjadi bisa dirasakan sampai sekitar 60 km dari lokasi musibah. Sementara musibah hampir serupa juga pernah terjadi di Desa Fanglin-masih di Provinsi Jiangxi-dan menewaskan setidaknya empat orang, Maret silam.

Sedangkan September lalu, 20 orang tewas di Cina tengah, setelah dua perusahaan kembang api liar meledak dan terbakar. Bulan lalu, dua wanita tewas, dan sejumlah orang lainnya luka-luka ketika sebuah pabrik kembang api liar di Mongolia, Cina Utara, meledak. (AP/AFP/Reuters/sha)
