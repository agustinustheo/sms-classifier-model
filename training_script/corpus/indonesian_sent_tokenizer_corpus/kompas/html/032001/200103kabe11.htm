 Kabel Listrik KRL Percikkan Api, Sejumlah Penumpang Cedera

Iwan
Jakarta, Kompas

Kabel listrik (pantograph) kereta rel listrik (KRL) dari Jakarta menuju Bogor, Senin (19/3), sekitar pukul 17.45, terputus akibat tersangkut kawat penghubung di bilangan Poltangan, Jagakarsa, Kecamatan Pasar Minggu, Jakarta Selatan. Percikan api akibat putusnya pantograph itu membuat penumpang KRL yang berada di atap terkejut dan berlompatan menyelamatkan diri sehingga sekitar 35 penumpang cedera.Menurut sejumlah saksi mata yang ditemui Kompas, penumpang langsung berloncatan begitu percikan api terjadi. Sebagian korban yang lompat maupun jatuh dari kereta dirawat di dua rumah sakit, masing-masing 15 korban di di Rumah Sakit Pasar Minggu dan 20 korban di Rumah Sakit Pasar Rebo.

Menurut Ruslan (22), mahasiswa Universitas Negeri Jakarta Jurusan Teknik Sipil yang tinggal di Cilendek, Kelurahan Cilendek, Kecamatan Bogor Barat, dia sendiri tidak berniat lompat saat itu.

"Tapi, karena didorong dari dalam, jadi saya jatuh. Mungkin karena panik melihat percikan api dari kabel membuat penumpang yang di pintu berlomba untuk lompat," tutur Ruslan. Dahinya dan sikutnya mengalami sobek.

Samsuri (30) warga Bojong Gede, juga berada di bagian pintu dalam gerbong ke lima. "Saya sendiri memang dengan sadar melompat walau kereta masih berjalan kencang. Sebab percikan bunga api dari kabel membuat saya berpikir mungkin kita yang di dalam juga akan kesetrum," tutur Samsuri, juga terluka pada dahinya.

Kepala Hubungan Masyarakat (Humas) PT Kereta Api Indonesia (KAI) Daerah Operasi I Zainal Abidin mengatakan, kecelakaan tersebut terjadi karena penumpang KRL melebihi kapasitas yang seharusnya. "Pantograph KRL nyangkut di bagian atas kawat. Seharusnya itu tidak terjadi," kata Zainal. Dia menduga, saat kereta menikung, sesaknya penumpang menyebabkan KRL miring sekali. "Saat itulah pantograph nyangkut," katanya.

Percikan api akibat hubungan pendek arus listrik menyebabkan sejumlah penumpang terkejut. Banyak di antara penumpang loncat dari gerbong.

Akibat kejadian di Pasar Minggu itu, hingga Senin malam hanya satu rel yang bisa difungsikan.

Menurut catatan Kompas, kejadian bermasalahnya listrik pada KRL itu pernah pula terjadi di Stasiun Pondok Kranji karena adanya hubungan pendek KRL jurusan Tanah Abang-Serpong, November 1988. Sebuah percikan api dengan bunyi ledakan di atap gerbong KRL memaksa penumpang berloncatan panik. (nic/ush)
