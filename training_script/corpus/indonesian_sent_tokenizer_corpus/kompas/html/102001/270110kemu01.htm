 Tabrakan KA di Rangkasbitung
Kemungkinan Ada Unsur Sabotase

Jakarta, Kompas

Berdasarkan penyelidikan sementara, tabrakan kereta api (KA) penumpang kelas ekonomi Jakarta-Rangkasbitung dengan kereta api batu bara di Rangkasbitung, Kamis (25/10) dini hari, disebabkan oleh faktor teknis. Rem kereta api tersebut blong, sedangkan katup pendistribusian angin ke gerbong belakang tertutup, sehingga rem tidak berfungsi. Meskipun demikian, Menteri Perhubungan Agum Gumelar mengatakan, apabila kecelakaan disebabkan "faktor X" atau sabotase, dia meminta aparat berwenang untuk memproses sesuai hukum.

"Jika ada indikasi ke arah itu, saya minta untuk diproses sesuai hukum yang berlaku," kata Menteri Perhubungan (Menhub) Agum Gumelar, di Bandara Soekarno-Hatta, Jumat sore, sekembalinya dari Kuala Lumpur menghadiri Pertemuan Tahunan Menteri-Menteri Transportasi ASEAN. Dalam kesempatan itu Menhub menjelaskan perkembangan kejadian tabrakan antarkereta api di Rangkasbitung yang menyebabkan tiga orang tewas dan 13 orang luka-luka tersebut, didampingi Dirjen Perhubungan Darat Susmono Soesilo dan Dirut PT Kereta Api Indonesia (KAI) Badar Zaenie.

Menteri Perhubungan meminta pihak-pihak terkait, termasuk polisi dan intelijen, melakukan penyelidikan mendalam tentang peristiwa itu. Namun, kata Agum, berdasarkan penyelidikan sementara di lokasi kejadian, terdapat fakta tuas pengereman rangkaian yang ada di lokomotif pada posisi emergency, yang menunjukkan masinis tidak bisa menghentikan kereta api secara normal saat tiba di Stasiun Rangkasbitung. Menyadari hal itu, masinis kemudian membunyikan suling lokomotif dan menghidup-matikan lampu sorot sebagai semboyan 39, yang mengindikasikan tanda bahaya.

"Mendengar semboyan tersebut, petugas yang berada di dua gerbong paling belakang menarik rem tanda bahaya," kata Menteri Perhubungan.

Ternyata rem tersebut juga tidak berfungsi. Padahal, dalam perjalanan sebelumnya dari Jakarta menuju Rangkasbitung dan berhenti di beberapa stasiun, semua rem berfungsi dengan baik. Karena rem tidak berfungsi dengan baik, kereta api penumpang tersebut menabrak kereta api batu bara yang sedang menunggu sinyal masuk.

Tidak wajar

Mengenai kemungkinan sabotase, lanjut Menhub Agum Gumelar, memang perlu diwaspadai karena dalam dua bulan terakhir terjadi tiga peristiwa yang tidak wajar. Peristiwa pertama tanggal 26 September 2001 pukul 20.05, ketika KA Parahyangan Jakarta-Bandung berhenti di Stasiun Cingur karena roda bogi terlilit kawat sling sepanjang delapan meter.

Selanjutnya, 13 Oktober 2001 pukul 18.36, KA Argo Gede Jakarta-Bandung sengaja diganjal rel sepanjang 4,2 meter di antara Stasiun Ciganea-Purwakarta, sehingga seluruh rangkaian kereta anjlok 14 as. Berikutnya, 18 Oktober 2001 pukul 01.50, KA Citrajaya jurusan Cilacap-Pasar Senen berhenti di antara Stasiun Sukatani-Ciganea karena sengaja diganjal bantalan kayu yang diletakkan melintang di atas rel. "Itu pasti bukan iseng, tetapi sengaja mau mencelakakan orang," kata Agum.

Menurut dia, mereka yang punya niat untuk melakukan sabotase diimbau untuk mengurungkan niat buruknya itu, karena yang rugi bukan hanya PT KAI, tetapi justru masyarakat. Selama masa Agum Gumelar menjadi Menteri Perhubungan, tercatat telah terjadi lima kali tabrakan KA dengan KA dan 15 kali tabrakan KA dengan mobil atau kendaraan bermotor serta menabrak orang, yang seluruhnya menelan korban jiwa 142 tewas dan 212 orang luka.

Normal kembali

Sementara itu, arus lalu lintas kereta api Jakarta-Merak dan sebaliknya, sejak Jumat pukul 04.45 sudah normal kembali. Rintang jalan Rangkasbitung-Jambu berhasil disingkirkan crane "Kumbokarno" yang didatangkan langsung dari depo Jatinegara.

Penumpang kereta api tujuan Merak sudah memadati Stasiun Tanah Abang, Kebayoran Lama, dan stasiun-stasiun lainnya yang dilintasi kereta api tersebut. Kecelakaan kereta api di Rangkasbitung sama sekali tidak menciutkan nyali sebagian besar masyarakat untuk menggunakan angkutan tersebut.

Kepala Humas PT Kereta Api Daerah Operasi I Jakarta Zaenal Abidin menambahkan, kereta api yang mengalami kecelakaan tersebut menarik enam rangkaian gerbong kereta dan tiga di antaranya rusak berat sehingga tidak dapat digunakan. Lokomotif kereta penumpang yang hancur akibat kecelakaan tersebut sudah diganti, tetapi gerbong penumpang tidak ada cadangannya, sehingga gerbong yang akan digunakan sementara hanya tiga gerbong kereta tersebut.
