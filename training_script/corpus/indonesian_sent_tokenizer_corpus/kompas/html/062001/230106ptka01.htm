PT KAI Rugi Rp 300 Juta per Hari

Kompas/dirman thoha
Semarang, Kompas

Rusaknya jembatan kereta api di Comal, Kabupaten Pemalang, Jawa Tengah (Jateng), Rabu (20/6) lalu, menyebabkan Daerah Operasi (Daops) IV PT Kereta Api Indonesia (KAI) mengalami kerugian Rp 300 juta per hari. Kerugian itu timbul oleh terganggunya perjalanan sejumlah kereta api jurusan Semarang-Jakarta yang nilainya 60 persen dari total pendapatan PT KAI Daops IV.

"Meski kerugian Daops IV cukup besar, hal itu sebenarnya tidak ada artinya dibanding dengan tidak terlayaninya penumpang kereta api, terlebih menjelang liburan kenaikan kelas tahun ini," ujar Kepala Daops IV PT KAI Makbul Sujudi yang dihubungi hari Jumat (22/6).

Makbul Sujudi menjelaskan, pendapatan dari kegiatan operasional PT KAI Daops IV rata-rata Rp 400 juta-Rp 500 juta per hari. Pendapatan itu diperoleh dari pengoperasian delapan rangkaian KA yang jadi andalan Daops IV. Kerugian setiap stasiun dari penjualan karcis penumpang rata-rata Rp 8 juta per hari.

Dalam perhitungan bisnis, akhir Juni yang bertepatan dengan liburan kenaikan kelas murid sekolah sebenarnya memberi keuntungan besar. Oleh karena itu, perbaikan tiang penyangga jembatan kereta api di Comal yang bersifat sementara digarap maraton supaya hari Jumat depan sudah bisa dilalui kereta api.

Menurut Kepala Daops IV, setelah mengoperasikan KA Argo Muria 1, mulai Jumat dini hari, perjalanan lancar. Perjalanan kereta api yang disambung bus dari Stasiun Pekalongan hingga Stasiun Pemalang memakan waktu 45 menit. Begitu juga dengan KA bisnis Fajar Utama yang diberangkatkan Jumat
pagi.

Kelancaran perjalanan semakin meyakinkan PT KAI Daops IV untuk memberangkatkan lagi beberapa kereta api lain. Seperti KA Tawangmas jurusan Semarang-Senen, Jakarta.

"PT KAI Daops IV sekali lagi minta maaf kepada pelanggan dan masyarakat yang selama ini setia menggunakan kereta api dengan adanya perubahan jadwal maupun pembatalan perjalanan karena rusaknya jembatan di Comal," ujar Makbul Sujudi.

Berhenti

Sampai Jumat sore, gerakan penurunan beton penyangga jembatan kereta api di Comal berhenti total. Posisi kedalaman amblesnya tiang beton penyangga jembatan mencapai 1,88 meter. Sewaktu uji coba lintasan menggunakan kereta lori kecil berbobot sekitar satu ton, penyangga jembatan tidak bergerak turun.

Stabilnya tiang beton penyangga jembatan memperlancar pemasangan tiang penyangga sementara. Dua tiang penyangga sementara akan dipasang di kiri kanan jembatan kereta api di tengah sungai. Pemasangan tiang penyangga berfungsi mencegah baja jembatan bergerak turun apabila tiang beton lama mengalami penurunan lagi.

Pemasangan tiang penyangga sementara memungkinkan jembatan kereta api di Comal bisa dilalui kereta api berbobot delapan ton lebih. Pemasangan ini diharapkan rampung, Jumat depan, sehingga jalur lintas pantura Semarang-Cirebon normal. Kereta api yang melalui jembatan di Comal diupayakan memperlambat kecepatan 5-20 kilometer per jam.

Setelah pemasangan tiang penyangga sementara berfungsi, akan dimulai pengerjakan proyek pembangunan tiang penyangga utama dari beton besar. Dua tiang penyangga utama ini dikerjakan selama enam bulan dan akan menggantikan tiang penyangga sementara maupun tiang beton lama. Sebagai upaya persiapan pembuatan tiang penyangga, tengah dikerjakan pembuatan bronjong batu di sekitar fondasi tiang beton lama.

Dari Stasiun Tawang diperoleh keterangan, tidak semua rangkaian KA dialihkan melalui selatan. Rangkaian yang dialihkan ke selatan melalui Solo, Yogya, Tegal, Cirebon, dan Jakarta hanya KA kelas ekonomi. Sedangkan KA eksekutif dan bisnis, sejak Jumat, tetap dioperasikan lewat utara.

Meski demikian, jumlah penumpang KA lewat utara rendah dibanding hari biasa. KA Argo Muria 1 dengan kapasitas 350 penumpang yang berangkat Jumat pagi dari Semarang mengangkut 186 penumpang. Bahkan, KA Argo Muria 1 dari Jakarta hanya mengangkut 86 penumpang dan dari Semarang sore harinya mengangkut 122 penumpang.

"Padahal, jumlah penumpang KA Argo Muria rata-rata 250 penumpang per hari. Ini mungkin disebabkan kurangnya informasi karena masyarakat menduga perjalanan lewat utara ditutup dalam jangka lama," ungkap Purwanto, staf Stasiun Tawang, Semarang.

Lain halnya dengan KA bisnis Fajar Utama dari Semarang yang mengangkut lebih dari 300 penumpang. Hal ini membuat petugas di Stasiun Pekalongan kewalahan karena waktunya bersamaan dengan datangnya KA Argo Muria 2 dari Jakarta di Stasiun Pemalang. (who/dth)
