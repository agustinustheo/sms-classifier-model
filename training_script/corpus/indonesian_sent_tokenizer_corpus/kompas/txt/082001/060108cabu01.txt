Cabut Izin Kepemilikan Senjata Api
Izin kepemilikan senjata api yang diatur beberapa undang-undang hendaknya dicabut atau ditinjau kembali. Seruan ini dilakukan karena izin itu, kini, dikhawatirkan berimplikasi negatif. Ada trend peningkatan kualitas kejahatan, ada kesan pamer kekuatan dari para pemilik senjata, dan ada kecemasan bangsa ini kelak akan menjadi bangsa koboi. Anak-anak sekolah dasar pun, kelak dikhawatirkan berkelahi atau bermain iseng dengan senjata api. Seruan perlunya pencabutan izin ini dikemukakan oleh usahawan dan mantan Menteri Perindustrian dan Perdagangan HM Jusuf Kalla, ahli hukum Aidir Amin Daud, praktisi hukum Ridwan J Silamma, usahawan dan pengamat masalah sosial Johannes Kitono, yang dihubungi terpisah, Minggu (5/8).  "Sebelum bangsa ini, atau kita semua sangat menyesal, sebaiknya izin itu dicabut sesegera mungkin. Amerika Serikat dan Filipina, dua dari banyak negara yang mengizinkan pemilikan dan jual beli senjata api, sudah sangat menyesal pernah mengeluarkan izin tersebut. Dua negara ini ingin menghentikan izin itu, tetapi ternyata sangat tidak mudah melaksanakannya," tutur Jusuf Kalla dengan nada masygul.  Wakil Kepala Badan Humas Markas Besar Kepolisian RI (Polri) Komisaris Besar Edward Aritonang yang dihubungi hari Minggu malam menyatakan, Markas Besar Polri tidak bisa melarang warga sipil memiliki senjata api karena undang-undangnya membolehkan. Polri hanya berhak memperketat persyaratan izin kepemilikannya. Sejauh ini pun, belum ada kolerasi antara kejahatan penyalahgunaan senjata api dengan izin kepemilikan senjata api pada warga sipil yang dikeluarkan Polri.  Menurut catatan  Kompas , undang-undang yang membolehkan warga sipil memiliki senjata api ialah Undang-Undang (UU) Nomor 8 Tahun 1948 tentang Pendaftaran dan Pemberian Izin Pemakaian Senjata Api, serta UU Drt Nomor 12 Tahun 1951 tentang Senjata Api dan Bahan Peledak.  Adapun UU yang menjadikan Polri berwenang mengeluarkan izin pemilikan senjata api adalah UU No 20 Prp Tahun 1960 tentang Kewenangan Perizinan yang Diberikan Menurut Perundang-undangan mengenai Senjata Api. UU Nomor 28 Tahun 1997 tentang Kepolisian Negara RI, yang di antaranya memberi kewenangan Polri mengeluarkan izin.  Kriminolog dari Universitas Indonesia Mulyana W Kusumah mengatakan, memang tidak mungkin melarang warga sipil memiliki senjata api karena UU membolehkan. Selain itu, memang ada warga sipil tertentu yang layak mempersenjatai diri karena jabatan yang diembannya. "Yang kini harus dilakukan, perketat pemberian izinnya. Hanya pejabat sipil tertentu yang bisa mendapat izin," kata Mulyana.  Pejabat sipil yang layak mendapat izin misalnya hakim dan jaksa. Anggota MPR/DPR dan pengacara, menurut Mulyana, tidak layak. "Pengacara bukan pejabat sipil. Anggota MPR/ DPR pun tidak dalam posisi menentukan kehidupan manusia, sedangkan hakim dan jaksa layak karena jabatan dan tugasnya sehari-hari yang determinatif (menentukan hidup manusia)," tuturnya.   Memahami bahaya   Jusuf Kalla menyebutkan, di AS, sejumlah anak-anak sekolah sudah menjadikan senjata api sebagai bahan mainan, dan kalau tidak puas dengan teman mainnya, bisa menembak dengan raut dingin. Di Filipina, kepemilikan senjata api itu ternyata sudah mengarah ke soal negatif. Senjata api menjadi instrumen untuk menekan. Lalu ada upaya menjaga diri dengan pengawal bersenjata, bom meledak di mana-mana, dan urusan culik-menculik sudah menjadi kebiasaan.  "Apakah aparat kepolisian kita menyadari hal ini? Apakah para pendidik bangsa memahami bahaya senjata api ini? Apakah para orangtua tidak cemas anaknya bermain senjata api, apakah para usahawan tidak gentar berusaha dalam sebuah negara yang iklimnya sudah buruk seperti ini?" papar Jusuf sambil menghela napas.  Soal tidak kalah peliknya, tambah Jusuf, jika muncul trend penculikan pengusaha terutama investor asing. Bisa-bisa tak ada investor asing menanamkan modalnya di sini. Kalau ini terjadi, posisi negara ini, yang sudah begitu lama dicekik krisis ekonomi, makin rawan.  Jusuf mengungkapkan, sebagai usahawan, ia mempunyai pergaulan cukup luas dan ia amat terkejut sebab pelbagai kalangan ternyata sudah memiliki senjata api itu. "Bukan cuma  lawyers  terkenal punya pistol, tetapi pengusaha dan bahkan anggota DPR. Negara kita seperti negara koboi saja. Wakil rakyat pakai jas, dan di dalam jasnya ada senjata api. Ini gejala apa?" sebut Jusuf.  Usahawan dan pengamat masalah sosial Johannes Kitono, ahli hukum Aidir Amin Daud, dan praktisi hukum Ridwan J Silamma sependapat bahwa langkah paling ideal untuk keluar dari persoalan ialah mencabut izin kepemilikan senjata. Usai pencabutan itu, para pemilik diminta menyerahkan senjatanya ke polisi. Langkah berikutnya, melakukan razia kepemilikan senjata api tanpa kecuali. "Razia itu semacam Operasi Sapu Jagat, Operasi Tertib," tandas Kitono.  Aidir Amin Daud dan Ridwan Silamma menyatakan, berbarengan dengan pencabutan izin itu, polisi dan aparat keamanan lainnya hendaknya bekerja lebih baik, untuk memberi rasa aman lebih baik kepada masyarakat. Sebagian di antara warga memaksa diri memiliki senjata api karena merasa tak aman, karena DKI Jakarta, misalnya, demikian rawan kejahatan.    Undang-undang   Edward Aritonang menyatakan, secara pribadi ia pun ingin hanya aparat keamanan yang memiliki senjata api. Namun, undang-undangnya menyatakan warga sipil diizinkan memiliki senjata api. Bahkan, sekarang ada wacana yang menginginkan untuk pejabat sipil tertentu harus dipersenjatai," tuturnya.  Menurut Aritonang, sampai Juli 2001, izin kepemilikan senjata api yang dikeluarkan oleh Markas Besar Polri hanya 1.100 buah, untuk seluruh Indonesia. Jumlah itu termasuk izin kepemilikan senjata api oleh anggota satuan pengamanan bank, polisi khusus kehutanan, dan aparat keamanan pada lokasi perkebunan tertentu.  Untuk warga sipil pun, lanjutnya, izin hanya diberikan untuk memiliki senjata api kaliber 22 sampai 32. Bukan senjata api standar yang digunakan Polri dan TNI. Sebelum membeli atau memiliki senjata api itu pun, warga sipil yang berminat harus lebih dahulu lulus beberapa tes, termasuk tes intelegensia dan tes kecerdasan emosional. Mereka pun hanya bisa membeli senjata api idamannya dari tujuh perusahaan yang telah ditunjuk pemerintah.  "Jadi, dia bukan membeli senjata api dulu baru minta izin. Namun, minta izin dulu, lalu menjalani berbagai tes. Kalau lulus, baru dia bisa beli. Tujuh perusahaan itu tidak akan menjual senjata api dagangannya kalau pembelinya tidak bisa menunjukkan surat izin dari Polri. Hanya tujuh perusahaan inilah yang berhak menjual, dalam rangka memperketat peredaran dan pengawasan senjata api," tutur Aritonang.  Menurut Aritonang, sejauh ini tidak ada korelasi antara dugaan adanya kecenderungan penyalahgunaan senjata api dengan jumlah izin senjata api yang dikeluarkan Polri. Aritonang yang tidak ingat secara rinci jumlah kasus kejahatan penyalahgunaan senjata api dengan tegas mengatakan, belum ditemukan kasus penyalahgunaan senjata api dilakukan oleh mereka yang memiliki izin. Juga belum ditemukan, senjata api yang berizin itu dipakai untuk kejahatan oleh bukan pemiliknya.  Namun, Aritonang mengakui, belum pernah mendengar vonis hakim yang relatif tinggi pada mereka yang menyalahgunakan senjata api. "Ini mungkin karena hukum kita hanya mengenal hukuman maksimal, serta kasus yang diajukan ke persidangan adalah pelakunya belum menggunakan senjata apinya itu, atau belum meminta korban jiwa," katanya.  Pasal 1 Ayat 1 Undang-Undang (Darurat) Nomor 12 Tahun 1951, Lembaran Negara 1951 Nomor 78 tentang Senjata Api dan Bahan Peledak, menyatakan,  Dihukum dengan hukuman mati, atau hukuman penjara seumur hidup, atau hukuman penjara 20 tahun bagi yang membuat, menerima, memperoleh, atau menyerahkan, menguasai, membawa, memiliki, menyimpan, mengangkut, menggunakan senjata api, amunisi, atau bahan peledak tanpa hak.
