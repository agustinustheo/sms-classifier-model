Stasiun Jatinegara-Senen Lumpuh Selama 16 Jam
Sebuah kereta api (KA) barang yang mengangkut batu bara anjlok di sekitar Stasiun Jatinegara, Jakarta, Rabu (22/8) malam sekitar pukul 22.30. Tidak ada korban luka-luka dalam peristiwa tersebut. Namun, akibat kejadian itu, jalur kereta api Stasiun Jatinegara-Stasiun Senen lumpuh selama 16 jam, serta seluruh perjalanan kereta rel listrik (KRL) yang melintasi Jatinegara dibatalkan.  Perjalanan kereta api tujuan Jawa Tengah dan Jawa Timur, baik kelas ekonomi maupun kelas bisnis yang berangkat dari Stasiun Senen, terpaksa memutar menggunakan jalur Senen-Tanah Abang-Jatinegara. Sedangkan kereta api lainnya mengalami keterlambatan lebih dari satu jam akibat padatnya jalur Gambir-Jatinegara.   Akibat dihentikannya perjalanan KRL, calon penumpang kereta api yang menuju Jakarta dari arah Bekasi, sejak Kamis pagi, terlihat menumpuk di beberapa stasiun, seperti Stasiun Bekasi, Klender, dan Kranji. "Saya terpaksa menumpang kereta dari Kutoarjo untuk menuju Jakarta," kata Setiawan (30), yang berangkat dari Stasiun Bekasi dan kemudian turun di Stasiun Gambir sebelum melanjutkan perjalanan menggunakan angkutan umum ke tempat kerjanya di sekitar Senen.   Satu kilometer   Menurut beberapa saksi mata, kereta pengangkut batu bara tersebut sudah anjlok sekitar satu kilometer sebelum memasuki Stasiun Jatinegara dari arah Senen. Roda sebelah kanan dari dua gerbong paling belakang keluar rel dan terus terseret tanpa diketahui masinisnya, M Husni. Kereta baru terhenti ketika rodanya terganjal wesel perlintasan rel di Stasiun Jatinegara.   Akibat anjloknya roda kereta tersebut, bantalan rel yang terbuat dari beton dan kayu mengalami kerusakan sepanjang lebih dari 200 meter terkena gesekan dengan roda kereta. Begitupun sejumlah wesel dan pengait bantalan rel yang terbuat dari baja patah sehingga harus diganti.  Kepala Humas PT Kereta Api Indonesia (KAI) Daerah Operasi I Jakarta Zaenal Abidin mengatakan, lalu lintas kereta api normal kembali hari Kamis sore sekitar pukul 15.00. Lamanya perbaikan ini karena tidak mudah mengangkat kereta anjlok yang bermuatan batu bara seberat 30 ton ditambah gerbong seberat 14 ton.  Apalagi, sambung Zaenal Abidin, kereta yang anjlok tersebut berada persis pada lintasan yang di atasnya terdapat kabel-kabel listrik untuk KRL sehingga operator  crane  harus bekerja sangat hati-hati. Selama pekerjaan dilakukan, aliran listrik terpaksa dimatikan sehingga KRL tidak bisa dioperasikan.  Sementara itu, Kepala Stasiun Senen, Besar Susmiarto, mengatakan, kereta api jarak jauh tujuan Jawa Tengah dan Jawa Timur yang berangkat dari Stasiun Senen tidak ada yang dibatalkan. Hanya saja, akibat lumpuhnya jalur Senen-Jatinegara, perjalanan kereta api dialihkan lewat Tanah Abang-Gambir-Jatinegara. Begitupun kereta api yang akan masuk Stasiun Pasar Senen dari Jawa Tengah dan Jawa Timur terpaksa memutar lewat Tanah Abang sehingga mengalami keterlambatan selama beberapa jam.
