Tiga Granat Aktif Ditemukan di Rel Kereta Api
Tiga granat aktif ditemukan diikat di bawah rel kereta api (KA) antara Stasiun Kedunggede dan Lemah Abang, Bekasi, Senin (8/1) sekitar pukul 14.00. Tepatnya pada KM 49,5. Granat tersebut, yang belum sempat meledak, ditemukan oleh Ahmad Taufik, seorang warga yang sedang menggembala ternak. Akibat kejadian itu, perjalanan 18 KA dari berbagai jurusan yang melintasi rel tersebut tertunda berjam-jam.  Penemuan granat di Desa Karangsari, Kecamatan Kedungwaringin, Kabupaten Bekasi, itu diawali oleh rasa curiga Ahmad Taufik terhadap rangkaian kawat yang terdapat di sekitar rel. Setelah ditelusuri, ternyata kawat itu tersambung dengan tiga granat aktif.   Dua granat diletakkan di celah besi rel KA dari jurusan Cikampek ke Jakarta. Satu granat lagi ditemukan di bawah bantalan rel KA ke arah yang berlawanan. Ketiga granat itu tersambung dengan kawat sepanjang kurang lebih lima meter.  Setelah menemukan granat itu, Taufik langsung melaporkan kepada penjaga perlintasan, Didi Junaedi. Didi segera memberhentikan KA dari Cikampek dan melaporkan penemuan granat itu kepada petugas stasiun terdekat, Lemah Abang. Petugas stasiun kemudian melaporkan kasus penemuan granat itu kepada aparat kepolisian. Granat kemudian berhasil diamankan oleh tim gegana Mabes Polri, pukul 16.50.   Kereta api   Menurut Kepala Humas Wilayah Daerah Operasi I Jakarta PT KAI Zainal Abidin, granat yang ditemukan itu belum sempat meledak. "Namun, ketiga granat itu diikatkan dalam sebuah kabel pada sebuah pohon. Kalau kereta api lewat, ada hentakan, bisa tertarik, dan bisa meledak," tambahnya.   Akan tetapi, lanjut Zainal, akibat adanya granat aktif tersebut, setidaknya 18 kereta dihentikan dan tidak melanjutkan perjalanan untuk sementara dari pukul 14.30 sampai pukul 17.20. "Dari arah timur, 12 kereta dan dari arah barat, 6 kereta," tuturnya.  Kereta api baru dapat dijalankan, ungkap Zainal, setelah granat diamankan dan dinyatakan aman oleh tim gegana. Tim gegana tiba di lokasi pukul 16.00. "Pukul 16.45, dua granat berhasil diamankan. Satu lagi sekitar pukul 16.45," tambahnya.   Pukul 17.08, rel kereta api dinyatakan bebas dari bahan peledak. Dengan demikian, sekitar pukul 17.20, kereta api baru dapat diberangkatkan.   Sementara itu, sampai pukul 22.00 Senin malam, kereta api Argo Lawu jurusan Solo yang seharusnya berangkat pukul 21.00 dan kereta api Argo Bromo Anggrek yang semestinya berangkat pukul 21.30 belum dapat diberangkatkan. Kereta api Argo pukul 22.00 masih berada di Depo Manggarai karena sedang dibersihkan. Kereta api itu terlambat tiba di Jakarta akibat adanya granat.  Zainal mengatakan, akibat penemuan granat itu, perjalanan kereta api memang sangat terganggu. Banyak penumpang yang mengeluh karena keterlambatan pemberangkatan atau tertunda perjalanannya. "Namun, setelah petugas di stasiun memberi penjelasan, para penumpang dapat memahami," tuturnya.  Sementara itu, Kepala Satuan Reserse (Kasatserse) Kepolisian Resor Bekasi Ajun Komisaris Besar Ilham Saparona mengatakan, pihaknya baru memeriksa lima saksi dari kasus penemuan granat itu. Selain Taufik, polisi memeriksa penjaga perlintasan Didi.   "Tiga saksi lain adalah warga sekitar yang melihat atau mengetahui peristiwa itu," tuturnya. Ia belum dapat menjelaskan bobot ledakan granat itu jika sampai meledak. "Itu wewenang tim gegana," katanya.
