KA Sawunggalih Anjlok, Perjalanan Kereta Terganggu

Kereta Api Sawunggalih jurusan Jakarta-Kutoarjo, Jumat (1/9) dini hari sekitar pukul 01.30 WIB, anjlok sekitar 200 meter selepas Stasiun Ciledug, Cirebon. Akibat anjloknya kereta api kelas bisnis tersebut, perjalanan kereta api dari Jakarta menuju Jawa Tengah dan Jawa Timur mengalami keterlambatan sekitar satu sampai dua jam.  Di Stasiun Gambir, perjalanan kereta rata-rata mengalami keterlambatan sekitar satu jam, baik yang akan berangkat maupun tiba di Stasiun Gambir. Penumpang menumpuk di berbagai sudut stasiun, namun penumpang umumnya tidak mengetahui penyebab keterlambatan ini.   Kepala Humas PT Kereta Api Indonesia (KAI) Daerah Operasi (Daop) I Jakarta, Zaenal Abidin Sabtu (1/9) mengatakan, jika anjlok di sekitar stasiun yang memiliki beberapa jalur rel, kecelakaan tersebut tidak akan terlalu mengganggu perjalanan kereta. Sebab, perjalanan kereta api bisa dialihkan melalui jalur lainnya.  Sedangkan Kepala Humas PT KAI Daop III Cirebon, Suhartono, mengatakan, belum mengetahui penyebab kecelakaan tersebut. Namun kereta yang anjlok tersebut sudah bisa diatasi Sabtu subuh sekitar pukul 04.30 WIB dan lalu lintas kereta api normal kembali. Kereta api yang pertama lewat setelah kecelakaan adalah KA Taksaka jurusan Jakarta-Yogyakarta, kemudian kereta api kelas eksekutif lainnya dan disusul kereta Sawunggalih.  KA Sawunggalih tersebut berangkat dari Stasiun Pasar Senen Jakarta, Jumat pukul 19.00 dan semestinya tiba di Kutoarjo, Sabtu pagi. Akibat kecelakaan tersebut, kereta api terlambat tiba di stasiun tujuan.
