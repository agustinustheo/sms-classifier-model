Diingat Saat Gunung Meletus


HINGGA saat ini, belum ada ilmu maupun teknologi yang mampu memastikan secara akurat, kapan satu gunung api akan meletus dan mendatangkan bencana bagi penduduk di sekitarnya. Karena itu, harus ada orang yang rela menunggui aktivitas gunung api tersebut dari detik ke detik supaya bisa memberikan tanda bahaya kepada penduduk.

Mengamati aktivitas gunung api ini merupakan profesi yang memerlukan kesabaran dan dedikasi cukup tinggi. Jumlah pengamat gunung api di Indonesia hanya 150 orang yang disebar pada 69 pos pengamatan gunung api. Namun, sebenarnya, gunung api yang aktif di seluruh wilayah Indonesia berjumlah 129.

Sebagai pengamat gunung api, mereka harus rela ditempatkan di daerah terpencil, hanya ditemani berbagai peralatan untuk mengumpulkan data-data dari aktivitas gunung api, alat-alat yang barangkali dingin dan beku di kedinginan gunung. Sebab, sebagian besar pos pengamatan terletak di daerah yang jauh dari pusat keramaian kota yang penuh dengan tempat hiburan.

Profesi ini terlihat sederhana, tetapi sebenarnya memerlukan pengorbanan yang cukup besar untuk mengumpulkan data-data mengenai aktivitas satu gunung api yang sedang diamati. Bahkan, seorang pengamat harus menyerahkan cintanya kepada gunung api yang diamati aktivitasnya agar tidak bosan dan benar-benar mengumpulkan data dengan cermat.

Darmo Lamane, pengamat gunung api dari Pos Pengamatan Gunung Api Gamalama di Pulau Ternate, bahkan mengatakan, ia "mengawini" gunung dahulu baru bertemu dengan istrinya. Ia juga mengatakan, hidupnya betul-betul diserahkan untuk tetap setia mengamati Gunung Gamalama sejak tahun 1985.

Tanggung jawab Darmo tidak sekadar ke atasannya, yang belum tentu datang ke pos pengamatan untuk melakukan inspeksi. Darmo mengatakan, ia toh tetap harus bertekun, harus melindungi penduduk di sekitar gunung. Jika mengetahui bencana sejak dini, bisa menekan jumlah korban jiwa dan kerugian material jika gunungnya meletus.

"Bagi orang yang senang mengamati kegiatan gunung api, mengikuti perkembangan aktivitas yang terdapat di perut gunung, merupakan hal yang sangat menyenangkan. Jadi bukan soal disiplin lagi, tetapi sudah merupakan kesenangan yang tersendiri," ujar Darmo.

Bahkan, pria asal Buton, Sulawesi Tenggara-yang ditemui usai menerima bantuan sepeda motor dari Menteri Energi dan Sumber Daya Mineral (ESDM) Purnomo Yusgiantoro di Pos Pengamatan Gunung Api Guntur, Garut, Jawa Barat-mengatakan, niat utama dari pengamat gunung api adalah melindungi penduduk dari bencana. Makanya, informasi mengenai aktivitas gunung api harus terus diikuti karena tidak ternilai harganya jika bisa mengingatkan penduduk untuk mengungsi daripada menjadi korban letusan gunung api.

TUGAS yang menempatkan pengamat harus menetap di kaki gunung api, menyebabkan pekerjaan mereka tidak terlalu dikenal oleh banyak orang. Jika suatu gunung meletus, barulah profesi ini disebut-sebut, karena mereka memang yang memberikan data terkini mengenai aktivitas gunung api yang sedang aktif-aktifnya.

Contohnya, dalam kasus meletusnya Gunung Papandayan yang terjadi saat ini, sebelumnya pos pengamatan gunung api yang berada di kaki gunung tersebut, selalu sepi dari tamu. Tetapi saat gunung ini meletus dan memuntahkan material dari perutnya, setiap hari dikunjungi wartawan yang datang mengambil data.

Bahkan, salah satu stasiun televisi nasional, sengaja memarkir mobil stasiun pemancar mereka di depan pos pengamatan api. Tempat kerja dari para pengamat gunung api, langsung menjadi kantor yang penting sebagai sumber berita.

Namun, menjalani profesi yang sangat dibutuhkan itu, tidak serta merta membuat para pengamat gunung api langsung hidup sejahtera dengan penghasilan yang memadai. Bahkan, mereka selalu khawatir dengan kesejahteraan di masa pensiun, jika harus turun gunung kembali ke kota.

Masalah kurangnya penghasilan petugas pengamat gunung api, diutarakan langsung kepada Menteri Energi dan Sumber Daya Mineral Purnomo Yusgiantoro oleh Edeh Suherman, petugas pengamat gunung api di Gunung Tangkuban Perahu, Jawa Barat. Edeh mengatakan, mohon pemerintah memberikan sedikit tambahan sumbangan kepada mereka, sebab penghasilan belum cukup untuk memenuhi kebutuhan mereka.

Selain itu, Edeh yang sebentar lagi pensiun, juga mengemukakan kerisauannya jika saat pensiun hanya mendapat surat penghargaan dari pemerintah Indonesia. Sebab, surat penghargaan tersebut, sama sekali tidak dihargai oleh masyarakat.

"Menjelang pensiun, saya ingin sampaikan, kalau bisa pengamat gunung api tidak usah diberi surat penghargaan karena tidak berlaku di masyarakat," ujar Edeh kepada Menteri ESDM. Ungkapan yang polos tersebut, mengisyaratkan para pengamat gunung api ini tidak punya persiapan pensiun karena sibuk dengan tugas sehingga tidak butuh sekadar penghargaan dari pemerintah.

Hal senada diutarakan Pudjihardjo yang sudah bertugas sejak tahun 1982 di Gunung Dieng. Tidak ada persiapan yang dibuat untuk menghadapi masa pensiun karena penghasilan yang tidak lebih Rp 1,5 juta tidak cukup untuk sehari-hari. "Jangankan masa pensiun nanti, sekarang saja kita masih pas-pasan," ujarnya.

Menurut Pudjihardjo, yang juga ayah dari lima anak, bekerja di tempat jauh dari kota, menyebabkan timbulnya pengeluaran ekstra yang tidak sedikit. Misalnya, untuk biaya sekolah anak-anak pengamat gunung api, selalu lebih mahal karena butuh biaya indekos di kota.

Padahal, seorang pengamat gunung api tidak memiliki kesempatan untuk mencari penghasilan tambahan karena tempatnya di daerah terpencil. Berbeda dengan pengawai negeri lainnya, bisa saja menjadi tukang ojek kalau memang gajinya masih kurang.

Belum lagi, menurut Pudjihardjo, pekerjaan sebagai pengamat gunung api memiliki risiko yang taruhannya nyawa karena harus mengambil data di dekat kawah, maupun langsung ke sumber gas beracun. Ia menambahkan, salah satu temannya yang bertugas di Gunung Bromo, akhirnya tewas karena menghirup gas beracun saat bertugas.

Namun, besarnya risiko yang dihadapi, profesi pengamat gunung api belum banyak diketahui masyarakat, bahkan juga oleh pemerintah. Khususnya menyangkut penghargaan pemerintah terhadap profesi ini, apa yang mereka terima dari pemerintah atas dedikasi dan pengabdian selama ini, masih terlalu kecil dibandingkan apa yang sudah diberikan kepada bangsa Indonesia.
