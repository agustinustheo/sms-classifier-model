Kembang Api Akan Semarakkan Jakarta

Detik-detik menjelang pergantian tahun, langit Jakarta akan semarak berwarna-warni. Sekitar 1.000 kembang api masing-masing seberat 1,5 - 12 kilogram akan diluncurkan ke udara dan meledak pada ketinggian sekitar 50-300 meter di sekitar Pantai Ancol Jakarta Utara. Pancaran bunga apinya akan berpencar pada radius 5-30 meter membentuk beragam variasi. 

Bola-bola kembang api ini masing-masing dipasang pada pipa pelontar besi sepanjang antara 60-150 cm sesuai ukuran besar bola api. Kembang api ini ditata di sepanjang 400 meter Pantai Ancol. "Paling bagus jika dilihat dari jarak 300-an meter," tambah Hendra Djaya, pimpinan Skylite Fireworks yang akan menjadi desainer atraksi kembang tersebut. 

Menurut Hendra Djaya, 600-an kembang api lontar dengan mesiu peledak yang akan digelar di Ancol tersebut adalah buatan PT Pindad. Sedangkan sekitar 400 kembang api hias, seperti yang bermodel air mancur diimpor dari Cina. "Kembang api hias ini tidak mengandung bahan ledak, dan aman sekalipun dipegang tangan ketika disulut sumbunya," jelasnya.

Atraksi kembang api digelar dengan komputerisasi. "Semua pelontar tersambung dengan kabel, makanya disebut sumbu elektronik. Konfigurasi waktu dan model yang dirancang desainer atraksi dioperasikan dengan komputer," papar Hendra Djaya. 

Meskipun atraksi kali ini tidak lepas dari upaya penghematan anggaran, Ancol menjanjikan atraksi kembang api yang spektakuler. 

MENGGELAR atraksi kembang api bukan pekerjaan sederhana dan tanpa risiko. "Makin lama bermain kembang api, saya makin memperhitungkan bahayanya," ujar Donny Rochyadi, Direktur Jakarta Special Effect yang lebih dikenal dengan sebutan Donny Kembang Api.   

Jenis kembang api yang dilontarkan dan meledak pada ketinggian tertentu bermula dari bahan kertas yang dipadatkan dengan mesin hidrolik. Bahan kertas ini dibentuk menjadi semacam mangkuk yang kuat, tetapi getas. 

Setelah diisi dengan campuran kimia berupa butiran warna yang disusun dengan tatanan tertentu, dua mangkuk disatukan membentuk bola. Tatanan butiran dalam bola inilah yang menentukan warna dan model kembang api yang dihasilkan ketika bola kembang api itu meledak di udara.

Ledakan di udara terjadi karena terbakarnya bubuk mesiu yang juga terdapat dalam bola kembang api. Bubuk mesiu yang ditambahkan pada bagian ekor bola berfungsi sebagai penyundut sekaligus pendorong yang melontarkan bola tersebut ke udara dari pipa atau laras pelontarnya. 

Sebagian bubuk mesiu yang lain menjadi sumbu yang dalam hitungan waktu dan ketinggian tertentu meledakkan bola kembang api itu. "Di bagian ini, membuat kembang api merupakan kerajinan tangan, tidak dibuat dengan mesin," ujar Hendra Djaya yang perusahaannya mengorganisasi Skylite Fireworks. 

Akan tetapi, kalau ingin melihat atraksi kembang api tersebut menghiasi langit Jakarta, lihatlah di tepi pantai sekitar Taman Impian Jaya Ancol (TPJA). Waktunya persis menjelang detik-detik datangnya tahun 2002.
