Palangkaraya Terkurung Api     Titik Api Meningkat di Sumatera

Kota Palangkaraya, ibu kota Kalimantan Tengah (Kalteng) sejak hari Rabu (10/7) terkurung api yang membakar hutan dan semak belukar di sekeliling pinggiran kota itu. Akibatnya, kabut asap makin pekat dan jarak pandang makin pendek. Hari Senin lalu, bahkan, jarak pandang paling jauh hanya 500 meter.  

Kabut asap juga masih menyelimuti udara Pontianak, Kalimantan Barat, namun kepekatannya tidak sampai mengganggu penerbangan di wilayah itu. Sementara itu, di Pulau Sumatera, jumlah titik api mulai meningkat dan terbanyak terdapat di Provinsi Riau. 

Demikian pemantauan Kompas dari lapangan, Rabu kemarin. Di kanan kiri ruas jalan Palangkaraya-Tangkiling dan Palangkaraya-Pulang Pisau sedikitnya terdapat 14 titik api yang membakar kawasan hutan dan semak belukar masing-masing sekitar 0,5 hektar. Sejauh ini, upaya pemadaman belum juga terlihat karena lokasi kebakaran berada sekitar satu kilometer dari ruas jalan raya. 

Keterangan yang dihimpun di Kelurahan Kalampangan menyebutkan, kebakaran itu berawal dari kegiatan petani setempat yang membersihkan lahan. Karena dikerjakan tanpa membuat pembatas, api merambat ke semak belukar sekitarnya hingga menjadi kebakaran hutan.  

Kepala Badan Pengelola dan Pelestarian Lingkungan Hidup Daerah (BPPLHD) Kalteng Sjahrani Sjahrin, kemarin, mengatakan, kondisi kebakaran hutan dan semak belukar di Palangkaraya sudah sangat kritis. Jika tidak ada tindakan pemadaman dalam waktu dua hari mendatang, Palangkaraya akan diselimuti kabut asap sangat tebal. Untuk mencegah agar tidak terjadi kebakaran hutan yang lebih luas, sekarang ini dibutuhkan dana operasional sekitar Rp 2,5 milyar.  

"Luas kebakaran di Kalteng sudah mencapai 50.000 hektar dan kami tidak bisa berbuat banyak karena sampai sekarang belum ada rapat koordinasi tentang penanggulangan bencana kabut asap dan kebakaran hutan. Kami masih menunggu petunjuk operasional dan tindakan terpadu," demikian Sjahrani. 

Selain mengganggu kesehatan, kabut asap yang semakin tebal itu dikhawatirkan akan mengganggu aktivitas penerbangan. Sejauh ini, jadwal penerbangan dilaporkan masih normal. 

Dari Pontianak, diperoleh keterangan, kabut asap masih juga menyelimuti udara kota khatulistiwa itu. Meski masih memedihkan mata, tingkat kepekatan asap dalam dua hari terakhir ini relatif lebih baik dari hari-hari sebelumnya. 

"Hari ini dan kemarin (Selasa dan Rabu-Red), semua traffic  sesuai jadwal. Baik keberangkatan maupun kedatangan tidak terganggu," kata Kepala Divisi Operasi Bandara Supadio, Pontianak, Kusasih Munir. Setelah pukul 08.00, katanya, jarak pandang pilot mencapai 10 km sehingga tidak mengganggu penerbangan. 

Juru bicara Pemerintah Provinsi Kalbar, Yakob Muchsin, berharap kondisi udara Pontianak terus membaik sehingga masyarakat dapat melakukan aktivitasnya dengan tenang. "Kami belum tahu kenapa dua hari ini udara relatif bersih. Mudah-mudahan ini terus membaik," katanya. 

Titik api di Sumatera 

Sementara itu, berdasarkan rekaman pemantauan satelit NOAA 12, Selasa petang, di seluruh Pulau Sumatera sudah terlihat tanda-tanda peningkatan jumlah hot spot (titik api). Jika tiga hari sebelumnya jumlah titik api masih di bawah sepuluh, pada hari Selasa itu jumlah titik api meningkat pesat menjadi 137 titik, dengan jumlah terbanyak di Provinsi Riau, kemudian menyusul di Sumatera Utara. Dari seluruh provinsi yang ada di Sumatera, berdasarkan pemantauan NOAA 12 hanya di Sumatera Selatan, Lampung, dan Bangka Belitung yang tidak terlihat adanya titik api. 

Berdasarkan pemantauan Kompas, melalui proyek Forest Fire Protection and Control Project (FFPCP) di Palembang kemarin, dari rekaman satelit NOAA di Sumatera diketahui di Provinsi Nanggroe Aceh Darussalam terekam satu titik api di sebelah utara Tapak Tuan. Di Provinsi Sumatera Utara terdapat lebih dari 20 titik api, terbanyak di sepanjang perbatasan dengan Provinsi Riau, tiga di sekitar Padang Sidimpuan, satu di sekitar Sibolga, dan dua di selatan Tanjung Balai. 

Di Provinsi Sumatera Barat terekam dua titik api, yaitu satu di sebelah utara Pariaman dan satu lainnya di wilayah perbatasan Kabupaten 50 Kota dengan Provinsi Riau. Di Provinsi Bengkulu yang biasanya jarang terlihat adanya titik api, berdasarkan rekaman Selasa petang, terlihat empat titik api di sekitar Kabupaten Bengkulu Utara. Sedangkan di Provinsi Jambi terlihat dua titik, yaitu di sekitar Sarolangun dan sekitar Merlung. Jumlah titik api terbanyak terekam di wilayah Provinsi Riau, terutama di kabupaten-kabupaten Rokan Ulu, Bengkalis, dan Siak. 

Meski titik api mulai banyak terlihat di Sumatera, sampai saat ini dari Riau belum dilaporkan adanya dampak kabut asap akibat mulai banyaknya titik api tersebut.
