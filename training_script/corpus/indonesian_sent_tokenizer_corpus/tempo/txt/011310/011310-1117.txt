Kereta Argo Gede Anjlok, 1.500 Penumpang Terlantar

Kereta Api Argo Gede KA-23 dari arah Bandung menuju Jakarta, sekitar pukul 19.40 WIB anjlok di kilometer 108 plus 2 per 3, Kampung Sulukuning, Desa Bunder, Kecamatan Jatiluhur, Purwakarta, Jawa Barat. Hal tersebut dikatakan, pegawai piket Stasiun Purwakarta, Yan Irianto, Sabtu (13/10) malam. 

Kereta api tersebut tertahan di antara stasiun Ciganea-Purwakarta. Selain itu, kereta api lain yang tertahan akibat kejadian tersebut adalah Parahiyangan K54 F dari Jakarta menuju Bandung dan kereta api Parahiyangan K55 dari Bandung menuju Jakarta. Kedua kereta api tersebut juga tertahan di Stasiun Cigane. 

Untuk mengantisipasi para penumpang yang terlantar, PT KAI memanfaatkan jasa angkutan kota dan bantuan lima bus dari Bandung. Mereka dibawa ke Stasiun Purwakarta. Sampai saat ini, para penumpang tersebut tampak bergerombol di Stasiun Purwakarta. 

Kereta Api Argo Gede memiliki enam gerbong sementara kereta api Parahiyangan memiliki tujuh gerbong. Diperkirakan, jumlah penumpang yang terbengkalai sekitar 1500 orang. Sementara itu, 30 anggota Polres Purwakarta melakukan pengamanan. Kepala Stasiun Purwakarta, Harris, hingga berita ini diturunkan belum bisa dihubungi.
