Keberangkatan KA Taksaka Lebaran Dibatalkan

Kecelakaan kereta api di daerah Brebes, Jawa Tengah, menyebabkan jadwal keberangkatan Kereta Api Taksaka Lebaran dari Stasion Gambir menuju Stasion Pasar Turi Surabaya pukul 09.45, dibatalkan. Hal itu dijelaskan Kepala Humas Deops I PT Kereta Api Indonesia, Zainal Abidin, kepada Tempo News Room di Stasiun Gambir, Selasa (25/12) siang.

Zainal menjelaskan, penumpang kereta api itu dialihkan ke KA Argo Dwipangga yang berangkat pukul 10.49 WIB. Argo Dwipangga ini juga mengalami keterlambatan selama dua jam. Seharusnya kereta api itu berangkat pukul 08.00 WIB. "Hanya itulah perubahan jadwal keberangkatan kereta, selebihnya kita berangkatkan sesuai jadwal," kata Zainal.
