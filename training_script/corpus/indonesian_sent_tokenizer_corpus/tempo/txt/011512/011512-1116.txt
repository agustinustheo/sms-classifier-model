Pemesanan Tiket KA Eksekutif di Stasiun Juanda Menurun

Satu hari menjelang Hari Raya Idul Fitri, arus pemesanan tiket Kereta api eksekutif di Pusat Reservasi Juanda tampak mulai menurun. Pengamatan Tempo News Room , Sabtu (15/12) siang, calon penumpang kereta yang antri hanya terdiri dari ratusan orang saja, dan tampak tidak terlalu berdesakan di pintu masuk.

Manajer Keuangan Pusat Reservasi Juanda, Nur Yasin Setiawan, mengatakan bahwa jumlah pengantri hari ini biasa-biasa saja. " Tidak seperti H-7 yang jumlah pengantrinya mencapai tiga ribuan orang," kata dia saat ditemui di ruangan pemesanan karcis. Ia menjelaskan, arus penumpang kereta api eksekutif relatif lebih lancar dan tidak terjadi kelebihan kapasitas. 

Pihak Kereta Api Indonesia (KAI) tidak menjual tiket berdiri untuk kelas eksekutif. Pihaknya telah memperbantukan tiga kereta api tambahan yaitu Kereta Api Gumarang dan Argo Bromo pagi, yang keduanya untuk tujuan Pasar Turi Surabaya dan kereta api Taksaka Lebaran untuk tujuan Yogyakarta. 

Setiap harinya, PT KAI memberangkatkan enam belas kereta api kelas eksekutif. Hari ini dari layar monitor daftar pemesanan tampak seluruh tempat duduk telah terisi penuh, kecuali dua seat lagi di kereta api Kamandanu tujuan Semarang Tawang dan lima seat di kereta api Taksaka Lebaran. "Umumnya seat yang tersisa hari ini karena ada pembatalan, misalnya tadinya yang memesan empat orang berangkat dibatalkan satu, sehingga yang tersisa itu pun sudah terpisah-pisah," kata Nur Yasin. 

Sabtu, jumlah penumpang yang akan diberangkatkan dengan kereta eksekutif dari Jakarta sebanyak 4.600 orang. Kereta api yang berangkat paling awal hari ini adalah Argo Muria, pukul 06.30. Sementara kereta api yang berangkat paling malam adalah Argo Bromo Anggrek Malam yang akan berangkat pukul 21.30. 

Untuk menghindari praktek percaloan, pihak PT KAI juga menerapkan peraturan bahwa penumpang yang akan berangkat adalah mereka yang namanya tertera di tiket. "Karena itu Alhamdulilah sampai sekarang belum ada kejadian percaloan di kompleks ini. Kalau di luar, saya tidak tahu," kata Nur Yasin.
