Kereta Api Pengangkut BBM Terbakar di Stasiun Wonokromo, Surabaya

Kereta api pengangkut bahan bakar minyak (BBM) terbakar ketika melintas di Stasiun Kereta Api Wonokromo, Surabaya, Sabtu (22/6) malam sekitar pukul 21.00 WIB. Dua gerbong kereta api yang berisi BBM terbakar habis, sedangkan 20 gerbong lainnya berhasil diselamatkan para petugas kereta api yang berada di lokasi kejadian.

Sebab-sebab kebakaran, sejauh ini belum diketahui. "Sumber api belum diketahui," ujar Kepala Humas Daerah Operasional (Daops) VIII Surabaya Sudarsono kepada Tempo News Room melalui telepon. Kereta api yang mengangkut pupuk dan semen milik Pertamina ini, kata Sudarsono, terbakar dalam perjalanan dari Surabaya menuju Madiun. 

Tidak ada korban jiwa maupun luka-luka akibat kebakaran itu. Sedangkan jumlah kerugian masih belum dapat ditaksir. Namun dokumen perjalanan mengenai barang-barang yang diangkut, berhasil diselamatkan.
