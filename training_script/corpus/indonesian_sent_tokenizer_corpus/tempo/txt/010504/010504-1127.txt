Kapolri Perintahkan Apel Senjata Api Secara Rutin

Kepala Kepolisian Republik Indonesia telah memerintahkan bawahannya untuk melakukan penertiban ke dalam dengan mengadakan apel senjata api dan amunisi secara periodik. Direncanakan, apel akan dilaksanakan satu bulan sekali. Bahkan bagi petugas operasional pemeriksaan senjata dikerjakan tiap hari Senin. Ini dilakukan setelah tertangkapnya pelaku jual beli senjata api dan amunisi di Depok, Selasa (3/4) malam. 
 
"Apel senjata api ini untuk mencegah kebocoran. Jika benar granat itu dari TNI dan Polri, hal ini merupakan kesalahan administrasi dan keteledoran polisi," cetus Kapuspen Polri Irjen Pol Didi Widayadi kepada pers di kantornya, Kamis (5/4) sore. Ia buru-buru menambahkan, keteledoran itu bukan disengaja polisi, tapi dimungkinkan adanya kebocoran gudang atau bisa jadi, saat latihan, ada oknum yang membawanya untuk dikumpulkan. 
 
Meski begitu, Didi mengakui, kontrol persejataan api milik TNI dan Polri memang lemah. Polri akan menindak tegas hal itu. "Kami harapkan jaksa dan hakim memberi sanksi maksimal jika terbukti ditemukan senjata api dan amunisi milik TNI dan Polri yang tidak pada tempatnya. Hal ini untuk mencegah kebocoran dan sebagai upaya preventif beredarnya amunisi lebih luas lagi," papar Didi. Para pelaku, tambahnya, dapat dikenai UU Darurat No 12 Tahun 1951 dengan ancaman hukuman mati. 
 
Didi mengaku prihatin dengan tertangkanya jaringan ini. Untuk itu, Polri menyikapinya dengan serius, di antaranya dengan mengadakan kunjungan ke Gudang Logistik Persenjataan di Cipinang, Kamis (5/4) pagi. Kunjungan itu untuk melihat distribusi senjata. Polri juga akan mengawasi ketat PT Pindad dan PT Wahana, yang dikenal sebagai distributor senjata api dan amunisi. 
 
Sementara itu, anggota Tim Gabungan Polda Metro Jaya dan Polres Depok berupaya melacak jaringan jual beli senjata dan amunisi setelah menemukan amunisi dalam jumlah besar dan narkotika pada malam penangkapan itu. "Barangkali ada sindikat narkotika yang juga bermain," tutur Kaditserse Polda Metro Jaya Kombes Pol Harry Montolalu pada pers, Kamis (5/4) siang. 
 
Harry menuturkan, dari penyelidikan sementara, para pelaku diperintahkan sponsornya untuk mencari dan mendistribusikan sendiri transaksi jual beli bahan peledak itu. Polisi, lanjutnya, saat ini sedang menelusuri siapa aktor utama dan pendistribusiannya sampai kemana. Apalagi, para pelaku mengaku telah terbiasa melakukan transaksi jual beli ini.
