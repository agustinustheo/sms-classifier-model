Kebakaran di Jakarta Timur, Api Masih Belum Bisa Dipadamkan

Api yang melalap pemukiman padat di Jalan Bendungan, Kelurahan Bali Mester, Jakarta Timur mulai pukul 16.30 WIB, semakin membesar. Api sudah mulai merembet ke perbatasan pemukiman padat dengan Kompleks Perumahan TNI AD di Jalan Jendral Urip Sumoharjo, Jakarta Timur. Petugas pemadam kebakaran masih berupaya memadamkan api. 

Lima unit mobil pemadam kebakaran sudah diturunkan di lokasi. Warga yang panik masih berupaya menyelamatkan barang-barang mereka. Ada yang membawa buntalan baju, televisi, kasur. Tapi banyak juga yang tidak sempat membawa apa- apa. 

Menurut keterangan seorang warga yang rumahnya sudah habis terbakar, Ny. Masmurah, 42 tahun, api berasal dari kompor yang meledak di sebuah warung nasi milik Usa, petugas Hansip setempat. Menurut Cecep, salah seorang tetangganya, saat kejadian, Usa sedang berjaga di pos RW. Sedangkan istrinya sedang memasak di rumah. "Saya baru saja habis salat Ashar, tiba-tiba saya mendengar orang berteriak: api ... api ... kebakaran ... kompor meledak," tuturnya pada Tempo News Room . 

 Cecep tidak menyangka api membesar hingga sulit dipadamkan. Dia langsung menuntun putri bungsunya yang buta, Maisaroh, 20 tahun, ke tempat yang aman. "Saya bawa dia ke sini (Jalan Jendral Urip Sumoharjo), tapi begitu saya hendak kembali, api sudah begitu besar dan rumah saya pun sudah habis dilalap api," ujarnya pasrah dengan mata sembab. 

Cecep bahkan tidak empat membawa dompet yang berisi identitas. Pakaianpun hanya tinggal yang melekat dibadannya saja. Salah seorang kerabatnya, hanya sempat menyelamatkan sebuah pesawat televisi miliknya. 

Sementara itu, api masih terus melalap pemukiman padat yang terletak di dekat sungai Ciliwung itu. Salah seorang anggta Koramil, E. Yogaswara, membenarkan, kalau api berasal dari kompor yang meledak di warung tersebut. "Jika angin masih terus berhembus kencang seperti ini dan terus mengarah ke kompleks TNI AD, tidak mustahil kalau perumahan ini juga akan terkena nantinya," tuturnya sambil sibuk meminta warga yang ramai menonton untuk menyingkir dan tidak berada terlalu dekat dengan lokasi kebakaran. 

Yogaswara memperkirakan sekitar 30-an rumah sudah menjadi korban di jago merah. Pemukiman padat tersebut, kata dia, umumnya ditinggali para pedagang kaki lima yang biasa berjualan di depan Lapangan Urip Sumoharjo. Hingga pukul 18.00 api masih belum bisa dipadamkan.
