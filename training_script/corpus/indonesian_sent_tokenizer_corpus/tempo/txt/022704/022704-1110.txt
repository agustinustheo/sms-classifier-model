Polda Metro Jaya Tangkap Dua Tersangka Pengedar Senjata Api

Jajaran Satuan Reserse Polda Metro Jaya berhasil menangkap dua orang tersangka pengedar senjata api. Dua tersangka ini ditangkap di tempat terpisah dengan barang bukti dua senjata api Colt 38 dan Colt 22, dengan masing-masing empat buah peluru. 

Kasat Serse Umum Polda, AKBP Raja Erizman, kepada wartawan di Mapolda Sabtu (27/4) menjelaskan, masing-masing tersangka bernama Partogi Sihotang ditangkap di daerah Pasar Induk Kramat Jati, Jakarta Timur, Rabu (24/4) ketika hendak menjual senjata api Colt 22 dengan empat buah pelurunya. 

Kemudian dari tersangka Partogi diperoleh keterangan bahwa senjata tersebut berasal dari seorang bernama Hengky yang kemudian juga berhasil di ringkus di lapangan Urip Sumohardjo, Jatinegara, Jakarta Timur, Kamis (25/4). Dari Hengky juga disita barang bukti pistol Colt 38 dengan empat buah pelurunya. 

Untuk menyelidiki peredaran senjata api yang meresahkan warga Jakarta ini, kata Erizman, pihaknya menugaskan beberapa orang anggotanya untuk melakukan transaksi dengan para pengedar senjata api tersebut. 

"Saat ini sedang dikembangkan, mungkin ada jaringan lain, dan kita akan kejar terus," tegas Erizman. Menurut dia, kedua tersangka ini masih sebatas pengedar di lapangan. Sementara pemilik atau atasan kedua tersangka belum diketahui. "Saat ini sedang dalam tahap penyelidikan," ujarnya. 

Tersangka Henky mengatakan, senjata api tersebut diperolehnya dari seseorang bernama Anto. "Anto itu memang pemain," ujar Henky yang berkepala plontos ini. Henky mengaku tinggal di Jatinegara dan sehari-harinya berdagang handphone. 

Sedangkan tersangka Partogi mengaku anak seorang anggota polisi berpangkat Kopka di Belawan, Medan. Ia mengaku baru kali ini berusaha untuk menjual senjata api. Itu karena ia yang pengangguran membutuhkan uang untuk hidup. Partogi berusaha menjual senjata api yang berasal dari Henky dengan harga Rp 2,7 juta.
