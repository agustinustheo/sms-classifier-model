Gerbong KA Eksekutif Gumarang Terbakar di Cepu

Gerbong salah satu kereta api eksekutif Gumarang Jurusan Gambir (Jakarta)- PasarTuri (Surabaya) terbakar, Sabtu (23/3), sekitar pukul 05.00 WIB. Humas PT Kereta Api Indonesia Daerah Operasi VIII/Surabaya Sudarsono, yang dihubungi Tempo News Room , membenarkan kejadian tersebut. "Diperkirakan akibat hubungan pendek," katanya.

Menurut Sudarsono, terbakarnya salah satu gerbong ini terjadi saat kereta api berada di antara Stasiun Wadu (Cepu) dan Kapuan. Hingga berita ini ditulis, kereta berhenti untuk melakukan perbaikan. Sudarsono sendiri mengaku belum mendapat laporan ini secara jelas. "Saya belum jelas. Saya sendiri mendapat laporan dari penumpang sekitar pukul 15.45 WIB," ujarnya.

Kejadian ini merupakan yang kedua kalinya selama Maret 2002. Sebelumnya, kereta api eksekutif Argo Dwipangga jurusan Jakarta-Solo juga mengalami musibah serupa, 4 Maret lalu. Api membakar bagian mesin pembangkit listrik saat kereta di Stasiun Karanggandul, Banyumas, Jawa Tengah.
