Sebagian Kereta Api Jalur Utara Batal Diberangkatkan

Bencana tanah longsor yang merusak jembatan kereta api di daerah Kahuripan, Ujung Negoro, Pekalongan, Jawa Tengah, mengakibatkan sebagian perjalanan kereta api yang melintasi jalur utara terpaksa dibatalkan. Hal tersebut dikatakan Humas PT KAI Daerah Operasi I Jabotabek, Zaenal Abidien kepada Tempo News Room , Minggu (17/2).

Menurut Zaenal, bencana tanah longsor akibat curah hujan yang tinggi itu terjadi sepanjang 25 meter, lebar 11 meter dan dengan kedalaman 10 meter. Perjalanan kereta api yang dibatalkan keberangkatannya, diperkirakan sebanyak 14 buah. Sebagian besar kereta api yang batal berangkat adalah kereta api tujuan Semarang. 

Sementara itu, kereta api tujuan Surabaya dari Stasiun Gambir seperti Argo Bromo dan Argo Bromo Anggrek akan dibelokkan ke jalur selatan, setelah memasuki stasiun Cirebon. Kereta itu akan melintasi jalur Yogyakarta dan Solo. Menurut Zaenal, banyak sekali penumpang yang terpaksa mengembalikan karcis atau menukarnya untuk keberangkatan di lain hari. 

Zaenal juga mengatakan, selain dibatalkan, beberapa kereta api juga digabung jadwal keberangkatannya. Kereta api Argo Bromo, misalnya, digabung dengan kereta api Argo Bromo Anggrek. Bahkan, apabila masih ada tempat yang tersedia, penumpang tujuan Semarang diperbolehkan menaiki kereta api lain ke Surabaya. 

Ketika ditanya tentang besarnya kerugian yang diderita oleh PT. KAI, Zaenal hanya mengatakan bahwa pihaknya belum usai melakukan pengecekan. Zaenal juga berharap bahwa dalam waktu 2 X 24 jam perjalanan kereta api akan normal kembali.
