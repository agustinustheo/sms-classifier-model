Enam Ratus Pendukung Gus Dur Sudah Pulang

Para pendukung Presiden Abdurrahman Wahid mulai Kamis (31/5) siang, sudah meninggalkan Jakarta. Mereka menggunakan jasa kerta api, sebagai salah satu alternatif untuk kembali ke asalnya. "Tadi siang sekitar 600 pendukung Gus Dur sudah pulang ke daerah masing-masing," kata Susmiarto, Kepala Stasiun Senen kepada Tempo di ruang kerjanya, Jakarta, Kamis (31/5) 

Mereka, kata dia, mulai berangkat dengan kereta api Gaya Baru Malang Selatan, pada pukul 12.00 Wib, menggunakan kereta Matarmaja jurusan Malang pukul 14.20, dan kereta api Gaya Baru Malang Utara jurusan Pasar Turi pukul 15.30, dan terakhir pada pukul 17.40 Wib dengan Kereta Api Kertajaya jurusan Pasar Turi. 

Menurut Susmiarto, gelombang kepulangan massa pendukung Gus Dur kemungkinan besar besok Jumat (01/6). Mereka akan berangkat mulai pukul 12.00 sampai pukul 17.40 Wib dengan jadwal kereta yang sama. Namun sampai saat ini belum ada koordinasi dari koordinator rombongan terhadap pihak Stasiun Senen. Menurutnya, apabila sampai pagi nanti belum ada koordinasi, mereka akan dianggap penumpang biasa. 

Meski demikian, lanjutnya, pihak kereta api masih bisa menampung pemulangan mereka. "Kita tidak mempersiapkan gerbong tertentu buat mereka, tapi pihak kereta api masih bisa menampung mereka. Ini kan kereta reguler," kata Susmiarto. 

Sutarto HS, Wakil Kepala Stasiun Senen, menambahkan bahwa pihak stasiun akan melihat jumlah mereka besok. Sebab menurut daftar yang ada pada pihak kereta api saat kedatangan mereka berjumlah sekitar 3.000 orang. "Mungkin di antara mereka sudah banyak yang naik bus, bahkan tadi sudah ada yang pulang 600 orang. Jadi jumlahnya kemungkinan tidak sebesar waktu saat kedatangan," kata Sutarto. 

Untuk mengatasi kemungkinan yang akan terjadi, kata Sutarto, pihak kereta api tetap melaporkan ke Polsek Senen apabila terdapat rombongan yang besar. Sampai saat ini suasana di Stasiun Senen masih berjalan normal, tidak ada antrian panjang dalam membeli tiket dari penumpang kereta api. Sementara beberapa aparat kepolisian dari Polres Metro Jaya dan Polsek Senen tampak berjaga-jaga di depan Stasiun Senen.
