Tarif Naik 40 Persen, PT KAI Optimis Tingkatkan Layanan

PT Kereta Api Indonesia sangat optimis bisa meningkatkan pelayanan kepada penumpang berkenaan dengan kenaikan tarif sebesar 40 persen pada 1 Juli 2002 dan 17 persen pada 1 Januari 2003 untuk kereta api kelas ekonomi. "Kenaikan ini lebih diarahkan untuk mengurangi beban subsidi dari pemerintah," Kata Humas PT KAI Patria Supriyoso di Jakarta, Kamis (27/6).

Menurut Patria, peningkatan itu tidak hanya sebatas layanan dasar semata. Layanan dasar tersebut meliputi standar minimum keselamatan, standar minimum fasilitas umum di stasiun dan di atas kereta api, standar minimum pelayanan kereta api jarak jauh serta pelayanan informasi. 

Menurut Patria, untuk mencapai titik impas biaya yang harus ditanggung oleh penumpang kereta api sebesar Rp 73,4 per kilometer. Saat ini, tarif yang berlaku baru mencapai Rp 31 per kilometer. Artinya, meskipun telah terjadi kenaikan total sebesar 57 persen pada 1 Januari 2003, angka tersebut tetap masih jauh dari titik impas. "Jadi, saat ini kami lebih menekankan pelayanan penumpang pada faktor keamanan," katanya. 

Untuk mencapai standar minimum pelayanan kereta api, PT KAI akan melakukan beberapa langkah seperti mencanangkan gerakan operasi kereta api tepat dengan menetapkan batas toleransi keterlambatan kereta api. Selain itu, PT KAI juga akan menyediakan kereta api cadangan untuk kereta api jarak jauh yang beroperasi bolak-balik. Operasi tertib angkutan (Terang) untuk menekan tingkat kebocoran pendapatan, juga akan diteruskan. "Setelah kami lakukan operasi Terang ternyata kami memperoleh peningkatan pendapatan yang cukup signifikan," kata Patria.
