Pekerja Kereta Api Tetap Tuntut Dirut PT KAI Mundur

Serikat Pekerja PT Kereta Api (SP KAI) tetap menutup Direktur Utama PT KAI Oemar Berto untuk mundur dari jabatannya. Suara ini tidak hanya dari pengurus pusat, tetapi juga kalangan pengurus daerah di Jawa dan Sumatera yang berkumpul di Bandung, Senin (18/2).

"Para pekerja kereta api tetap menuntut Dirut PT KAI Oemar Berto untuk turun," ujar Sekretaris Umum SP PT KAI Amin Abdurrachman dalam keterangan resmi seusai rapat konsolidasi di Bandung tadi siang. Bagaimana dengan ancaman mogok? Menurutnya, SP KAI akan menjadikan opsi tersebut untuk melakukan tawar-menawar jika aspirasi tersebut tidak digubris. "Kami sebetulnya tidak ingin mogok," ujar Amin.

Menurutnya, rapat tersebut bertujuan untuk konsolidasi. Ia juga menjabarkan beberapa alasan dibalik tuntutan mundur itu. Pertama, kredibilitas Berto diragukan. Kedua, selama meniti karir, Oemar Berto selalu dianakemaskan oleh penguasa kereta api saat itu. "Contohnya, dia tidak pernah menjabat kepala sub seksi, tiba-tiba menjadi kepala sub direktorat. Lopatan karir itu tidak sesuai peraturan di PT KAI," ujar Amin.

Soal ketiga, Oemar Berto dinilai gagal dalam tugasnya sebagai Direktur Pengembangan Umum Usaha PT KAI, menyusul tragedi kecelakaan kereta api di Brebes, Jawa Tengah, tempo hari. Amin juga membantah SP KAI tidak kompak lantaran opsi mogok kerja tidak didukung organ-organ di daerah. Ia mencotohkan gerakan reformasi 1998 oleh mahasiswa. "Yang dekat dengan pusat bisa memukul langsung ke pusat ke kekuasaan. Kalau yang di daerah menunggu informasi."

Pernyataan Amien dibenarkan sejumlah pengurus daerah. Misalnya Ketua Pengurus Daerah II SP KA Cirebon, Apid Suryadi. "Kita masih kompak," katanya. Pekerja kereta api di wilayah Cirebon siap menunggu instruksi dari pusat. "Jika pemogokan terjadi di Cirebon, kondisi perekonomian dan politik akan terganggu. Pasalnya, Cirebon termasuk jalur padat dan vital terutama untuk mengangkut bahan bakar ke berbagai daerah," ujarnya.
