KA Tabrak Kijang, Seorang Tewas

Kepala Humas PT Kereta Api Indonesia Daerah Operasi I Zaenal Abidien membenarkan adanya tabrakan kereta api (KA) dengan mobil Toyota Kijang yang mengakibatkan seorang tewas, Sabtu (25/5). Akan tetapi, Zaenal mengaku tidak tahu apakah pintu lintasan kereta api telah ditutup ketika terjadi tabrakan di Pasar Minggu, Jakarta Selatan tersebut. "Saya nggak ngerti persis ditutup atau tidak," kata Zaenal kepada Tempo News Room, Sabtu (25/5) malam. 

Zaenal mengaku belum memperoleh laporan mengenai kecelakaan yang terjadi di di lintasan pintu kereta api Pasar Minggu tersebut. Dalam kecelakaan tersebut, sebuah mobil Toyota Kijang yang mogok tertabrak kereta Jabotabek yang melaju dari Stasiun Kota, Jakarta. Mobil tersebut terbakar dan bunga api sempat terlihat di tubuh kereta sarat penumpang itu.

Peristiwa tabrakan ini bermula, ujar Zaenal, ketika mobil tersebut melintasi pintu perlintasan kereta Pasar Minggu. Ketika berada di tengah lintasan, mobil tersebut mogok. Sebelum semua para penumpang sempat keluar dan menjauhi mobil tersebut, kereta api ekonomi jurusan Jakarta-Bogor yang baru berangkat dari stasiun Pasar Mingu langsung menabrak mobil naas tersebut. 

Menurut Zaenal, jalur lintasan kereta api Jakarta-Bogor ini merupakan jalur lintasan terpadat setelah jalur Jakarta-Cikampek. "Tiap lima menit ada kereta yang lewat," katanya. 

Zaenal juga mengatakan, peristiwa kecelakaan seperti ini sulit dihindari karena letak jalan raya dan lintasan kereta api berada dalam bidang yang sama. Untuk itu ia berharap, "Agar tidak sebidang, bisa dengan fly over atau by pass ." 

Tabrakan kereta api itu sendiri berjalan sangat cepat. "Penjaga maupun (masinis) kereta tidak bisa berbuat apa-apa," kata Zaenal.

Dalam peristiwa yang terjadi pada pukul 18.07 ini, seorang anak kecil yang menjadi penumpang mobil meninggal dunia. Sementara itu, dua orang luka berat dan delapan orang lainnya luka ringan. Para korban memperoleh perawatan di Rumah Sakit Pasar Minggu, Jakarta. 

Untuk para korban, kata dia, ada sejumlah dana yang akan diserahkan oleh pihak PT Kereta Api, namun dia mengaku tidak mengetahui berapa persis jumlahnya.
