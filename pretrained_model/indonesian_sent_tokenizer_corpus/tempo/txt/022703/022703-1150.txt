Jaksa Optimis Kesaksian Edi dan Bahrum Tak Patahkan Dakwaan Tommy

Jaksa Penuntut Umum Andi Rahman Asbar optimis kesaksian Ajun Komsaris Besar Polisi Edi Purbo dan Ajun Komisarsi Polisi Bahrum Hasibuan dalam sidang kasus pembunuhan, pemilikan senjata api dan melarikan diri tidak akan meringankan terdakwa Hutomo Mandala Putra alias Tommy Soeharto. "Tidak ada pengaruhnya meski Edi dan Bahrum tidak membawa surat penyitaan dari pengadilan saat menggeledah Apartemen Cemara," kata Andi usai sidang di Pengadilan Negeri Jakarta, Rabu (27/3).

Menurut Andi, kedua polisi Polres Jakarta Pusat itu tidak menyalahi prosedur penggeledahan dan penyitaan senjata api di Apartemen Cemara pada 5 dan 6 Agustus 2001. "Itu tidak cukup kuat untuk mematahkan dakwaan," katanya. Padahal dalam sidang, kedua saksi ini dicecar pelbagai pertanyaan seputar tidak adanya surat ijin penggeledahan dan penyitaan dari pengadilan oleh kuasa hukum Tommy Soeharto. 

Tanya jawab antara OC Kaligis dan Bahrum, misalnya, sempat dihentikan oleh hakim ketua Amiruddin Zakaria karena terlalu bertubi pertanyaan dan jawaban saksi. Andi beralasan, usai penggeledahan pada tanggal 7 Agustus kedua anggota polisi itu meminta surat penyitaan. Menurut Andi, hal itu sah saja, karena polisi yang berencana menangkap Tommy malah menemukan beberapa pucuk senjata api berikut pelurunya. 

Alasan Edi sendiri tidak mempunyai surat ijin karena kondisi yang mendesak. Sementara itu, kuasa hukum Tommy, Juan Felix Tampubolon optimis jika tidak adanya surat ijin penyitaan membuat sitaan menjadi tidak sah. "Tidak hanya akan meringankan, tapi itu membuktikan penyitaan dan penggeledahan tidak memenuhi syarat," kata Juan Felix usai sidang. Soal tidak adanya sidik jari Tommy di senjata-senjata api tersebut, Andi juga tidak mempermasalahkan. 

"Itu tidak ada hubungannya," katanya. Tommy sendiri, dengan tidak ditemukannya sidik jari dirinya menyangkal telah memiliki senjata api tersebut. Edi dan Bahrum dalam sidang itu mengatakan bahwa Hetty Siti Hartika, pengelola Apartemen Cemara, menyebutkan bahwa senjata-senjata api itu milik Tommy. Bahkan Hetty, saat ditanya keduanya, mengaku Tommy sering datang ke Apartemen Cemara dan tidur di ruangan yang di dalamnya ditemukan senjata api.
