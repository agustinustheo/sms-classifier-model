Poltabes Makassar Tangkap Mahasiswa Penjual Senjata Api

Kepolisian Kota Besar (Poltabes) Makassar, Sulawesi Selatan menahan dua mahasiswa Universitas Negeri Makassar (UNM), Emil dan Iswan, sejak Jumat (21/6). Keduanya disangka memiliki dan memperdagangkan senjata api rakitan. 

Seorang mahasiswa lainnya, Iwan dan Kifli yang bukan mahasiswa, diduga terlibat dalam kasus ini masuk dalam Daftar Pencarian Orang (DPO). Polisi menyita pistol rakitan jenis FN beserta peluru kaliber 5,3 mm. 

Kepala Satuan Reserse Poltabes Makassar, Ajun Komisaris Hambali, yang ditemui di ruang kerjanya, Sabtu (22/6) mengatakan keempat tersangka itu diancam pidana maksimal 20 tahun penjara atau hukuman seumur hidup. Menurut dia, pihaknya sudah menyiapkan surat penangkapan untuk Iwan dan Kifli. Kedua tersangka tersebut diminta menyerahkan diri karena menjadi target operasi. 

Selain keempat tersangka itu, aparat kepolisian menduga beberapa orang lainnya yang juga memiliki senjata api rakitan. Tapi tersangka Emil berjanji menunjukkan tempat-tempat senjata api rakitan di kampusnya. Menurut Hambali, Kifli dan Iwan juga merupakan kunci untuk membongkar jaringan pembuatan senjata api rakitan. 

Kasus itu terbongkar lantaran keisengan saksi Anto yang berniat membeli senjata api. Keinginan itu disampaikan kepada Iswan yang kemudian menghubungi Iwan. Setelah mendapatkan barang itu, Iswan mengantarkan barang yang dijual dengan harga Rp 150 ribu ke rumah Anto. Tapi, lantaran Anto sedang tidak ada di rumah, Iswan membawa kembali pulang senjata itu. Di perjalanan pulang itulah ia langsung disergap tim Reserse Mobil Brimob Polda Sulsel, yang sudah mengintainya. 

Kepada penyidik Iswan membantah tuduhan bahwa ia pemilik senjata api rakitan itu. Dia mengaku hanya meminjamnya dari Iwan untuk berjaga-jaga diri. Pasalnya, ia hendak menagih piutang dari Anto sebesar Rp 250 ribu. Ia mengaku tidak pernah mencoba senjata api itu sebelumnya. "Pistol itu saya pinjam dari teman sekampus saya untuk menakut-nakuti saja," akunya kepada penyidik.
