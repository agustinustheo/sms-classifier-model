Agum Gumelar Bela Kereta Api

Menteri Perhubungan (Menhub) Agum Gumelar tidak terima kereta api dipersalahkan berkait kecelakaan maut di Serang kemarin, yang menewaskan 20 warga Kampung Kurus, Cilincing, Jakarta Utara. "Dalam hal ini, kereta api tidak bersalah. Tetapi saya akan menindak setiap kesalahan yang dibuat operator," kata Agum kepada wartawan usai mendampingi Wakil Presiden Hamzah Haz dalam peringatan Hari Puspa dan Satwa di Jakarta, Senin (5/11). 
 
Pembelaan itu dilakukan Agum dengan berdalih, menurut undang-undang, kereta api harus didahulukan. Selain kereta api, alat transportasi yang perlu diprioritaskan pengguna jalan adalah ambulan dan mobil VIP. Karena itu jika terjadi kecelakaana kereta api denga bus, kereta tidak bisa disalahkan. "Kan tidak bisa kereta api berhenti untuk melihat kendaraan yang lewat," kata dia. 
 
Tapi, bagaimana bila di lintasan rel-jalan raya tidak dilengkapi palang pintu pengaman seperti terjadi di lokasi kecelakaan, kawasan Kabaharan Dukuh, Serang, dimana pengguna jalan tidak tahu kereta lewat? Agum hanya berujar rendah, bahwa harus dilakukan penertiban terhadap lingkungan yang kurang kondusif. "Saya minta masyarakat khususnya sopir bus agar mengikuti petunjuk yang ada," ujar Agum.
 
Dia mencontohkan pemasangan PT KAI terhadap rambu-rambu peringatan atau polisi tidur, sebagai ganti bila tidak ada pintu pengaman di persimpangan rel kereta-jalan raya. "Sayangnya, cukup banyak masyarakat yang tidak mematuhi seperti dilanggarnya lampu merah. Jadi pemahaman masyarakat juga harus ditingkatkan," tuturnya.
