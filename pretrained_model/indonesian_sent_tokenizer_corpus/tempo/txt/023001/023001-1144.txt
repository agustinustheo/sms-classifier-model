Rel Amblas di Ciganea Kereta Api Terlambat Satu Jam

Jalur perjalanan kereta api di kawasan Ciganea, Purwakarta, terganggu karena tanah penyangga rel amblas sepanjang 10 meter menyusul hujan deras yang mengguyur Jawa Barat kemarin malam. Akibatnya, perjalanan kereta api yang melintas di daerah tersebut, Rabu (30/1), mengalami keterlambatan sekitar satu jam karena perbaikan oleh petugas.

Humas PT Kereta Api Indonesia, Gatot Wibowo, mengatakan tidak ada satu pun jadwal pemberangkatan kereta yang dibatalkan. "Tapi, 24 kereta api mengalami keterlambatan berangkat sekitar satu jam sejak kereta pertama yang berangkat pukul 05.00 wib," ujar Gatot kepada Tempo News Room tadi sore di Bandung. 

Selain di Ciganea, perjalanan kereta api juga terganggu di lintasan Bandung-Rancaekek karena banjir bandang di Rancaekek. Meski begitu, tidak terjadi pembatalan kereta pada jalur yang tiap harinya melayani 10 ribu penumpang ini. Keterlambatan terjadi sekitar dua jam karena kereta tidak bisa berjalan dengan kecepatan normal.

Gatot juga mengungkapkan, gangguan perjalanan kereta juga terjadi di Medan, Sumatra Utara. "Gangguan sangat serius karena mengakibatkan terhentinya operasi angkutan ini selama dua hari untuk jalur Medan-Tanjung Balai," ujarnya. Akibatnya, jelas Gatot, kereta api tidak bisa melayani pemberangkatan sekitar 600 penumpang.
