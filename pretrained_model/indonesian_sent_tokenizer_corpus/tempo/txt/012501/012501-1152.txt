Polisi Tangkap Pemilik Senjata Ilegal

Tempat transaksi senjata api ilegal di wilayah Kebayoran Baru, Jakarta Selatan, 20 Januari lalu telah diendus aparat kepolisian. Polisi pun berhasil menangkap pelaku transaksi senjata gelap itu.. "Polisi sudah lama mengendus," kata Kadispen Polda Metro Jaya Kombespol Anton Bachrul Alam, didampingi Kaditserse Polda Metro Jaya Kombespol Harry Montolalu, di ruang Ditserse Polda Metro Jaya, Kamis (25/1) siang.

Penangkapan ini dilakukan oleh satu tim reserse mobile yang dipimpin Ipda Amir Hadi. Mereka berhasil meringkus Stephanus Turangan, (36 tahun), seorang karyawan yang tinggal di jalan Tulodong Bawah B-17 Kelurahan Selong, Kebayoran Baru Jakarta Selatan. Di alamat itulah transaksi itu terjadi. "Tersangka ditangkap karena membeli senjata api ilegal," ujar Anton. 

Menurut Kadispen polda, Stephanus hanya membeli, menyimpan, dan memiliki. Polisi pun belum melihat adanya indikasi kejahatan dari dirinya. Dari pengakuan tersangka, kata Anton, diketahui pula bahwa Stephanus adalah anggota salah satu klub menembak. Namun tersangka menyangkal keanggotaannya di PERBAKIN.

Dalam penangkapan itu, polisi berhasil menyita sepucuk senjata api laras panjang, sepucuk pistol merek Smith and Wesson, sepucuk pistol merk Heckler & Koch, dua pucuk pen gun , 90 butir peluru kaliber 5,56 mm, 50 butir peluru kaliber 22, sebutir peluru cahaya, empat kotak peluru 9 mm buatan PT Pindad, 10 peluru hampa, sekotak alat pembersih merek Pomsiok, dan satu sarung pistol untuk senjata pinggang. "Meski kecil, pen gun ini harganya jutaan rupiah dan dapat mematikan," kata Harry.

Menurut tersangka yang ikut dihadirkan dalam konferensi pers itu mengatakan bahwa alasan dia membeli senjata api itu untuk pertahan diri. Namun ia tidak mau mengurus ijin kepemilikan senjata api karena takut ketahuan petugas. "Senjata api ini saya beli dari seorang bernama Sandy," ujarnya. Sandi sampai saat ini masih dalam pengejaran polisi.

Sebelumnya, polisi juga melakukan penggerebekan di jalan Raya Bogor. Penggerebekan di depan broker transaksi jual beli senjata api seharga Rp 2.500.000,00 itu dilakukan satu tim Panah 86 dipimpin Ajun Komisaris Polisi G. Nasution (14/1). Di situ polisi berhasil menagkanp Heru Taruna bin Suryadi. Setelah mengembangkan kasus tersangka, polisi berhasil meringkus Ericos Monanza di daerah Ciracas sehari sesudahnya.

Polisi berhasil menyita barang bukti berupa sepucuk pistol merk Colt kaliber 22. Selain itu, polisi berhasil mengetahui lokasi perakitan senjata api di daerah Cipacing, Bandung. Kasus ini melibatkan tersangka bernama Pipin. "Kasus ini telah ditangani Polda Jabar," ujar Kadispen.
