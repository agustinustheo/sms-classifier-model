Jalur Kereta Api Jakarta-Bandung Normal Kembali

Setelah sempat lumpuh total sejak Sabtu (13/10) sore akibat anjloknya kereta api Argo Gede di daerah Purwakarta, Jawa Barat, jalur kereta api Jakarta-Bandung, Minggu (14/10) pagi pukul 09.20 WIB sudah kembali normal. "Pukul 9.20 pagi ini sudah berangkat Kereta Api Parahyangan pertama dari Stasiun Gambir menuju Bandung," kata Zaenal Abidien, Kepala Hubungan Masyarakat PT Kereta Api Indonesia kepada Tempo News Room melalui saluran telpon seluler, Minggu (14/10) siang. 

Menurut Zaenal, penyebab anjloknya kereta api Argo Gede jurusan Bandung menuju Jakarta pada pukul 18.36 WIB kemarin (13/10) di Purwakarta itu akibat adanya potongan-potongan rel di atas lintasan rel kereta api. Dampaknya, tadi malam dua kereta api dari Stasiun Gambir, satu kereta api dari Stasiun Senen dan satu dari Stasiun Bandung mengalami pembatalan keberangkatan. Sedangkan pagi hari ini, satu kereta api dari Jakarta dan satu kereta api dari Bandung ikut mengalami pembatalan. "Untung pukul 8.20 pagi ini kereta sudah di rel, pukul 8.30 sudah ditarik ke Bandung."

Selain itu, akibat pembatalan ini menurut Zaenal, sebanyak 600 karcis yang sudah dipesan Sabtu dan 500 karcis pada Minggu dipulangkan oleh semua calon penumpangnya. "Jadi totalnya kerugian PT KAI dalam dua hari ini sekitar Rp 30 juta," katanya. 

Yang lebih merugikan dari kejadian ini, masih menurut Zaenal, adalah semakin buruknya citra PT KAI di mata masyarakat. "Padahal semua personil sudah berusaha bekerja keras dan maksimal, dan semua sarana dan prasarana baik semuanya," kata dia. Zaenal menambahkan, berbagai kejadian dalam sebulan terakhir lebih disebabkan faktor alam. Memang pekan lalu, sebuah kereta api Parahyangan juga anjlok di daerah Purwakarta akibat musim hujan yang mengakibatkan labilnya kontur tanah tempat lintasan rel. 

Zaenal tidak mengetahui mengapa ada potongan rel di atas rel kereta api. "Potongan rel tersebut sudah dibawa ke polisi dan sekarang Polres Purwakarta sedang mengusut kejadian ini. Saya tidak tahu penyebabnya, tapi banyak yang menduga ini sebuah sabotase," katanya. Sebelumnya, tadi malam (13/10) Kapolres Purwakarta Ajun Komisaris Besar (Pol) Elan Sugilan mengatakan ada indikasi sabotase dalam peristiwa anjloknya rel kereta api di Purwakarta ini, setelah ditemukan potongan rel sepanjang 4,2 meter dengan posisi terbalik. 

Mengantisipasi berbagai kejadian tersebut, Zaenal dan jajarannya akan melipatgandakan kewaspadaan dan memperketat pengawasan di lintasan rel kereta api. Tapi dia pesimis usaha ini dapat mengurangi tingkat kecelakaan kereta api, jika tidak ada partisipasi masyarakat dalam membantu PT KAI. "Bukannya malah membantun kami, selama ini masyarakat hanya mau memanfaatkan lahan yang ada di sepanjang pinggiran rel kereta api," katanya setengah mengeluh.
